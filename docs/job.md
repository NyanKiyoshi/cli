## saleor job

```
saleor job [command]

Commands:
  saleor job list  List jobs

Options:
      --json     Output the data as JSON                               [boolean]
  -V, --version  Show version number                                   [boolean]
  -h, --help     Show help                                             [boolean]
```

### saleor job list

```
saleor job list

List jobs

Options:
      --json     Output the data as JSON                               [boolean]
      --env                                                             [string]
  -V, --version  Show version number                                   [boolean]
  -h, --help     Show help                                             [boolean]
```
