## `saleor checkout` command

```
saleor checkout [command]

Commands:
  saleor checkout deploy [name]  Deploy `saleor-checkout` to Vercel

Options:
      --json     Output the data as JSON                               [boolean]
  -V, --version  Show version number                                   [boolean]
  -h, --help     Show help                                             [boolean]

```
