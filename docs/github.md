## `saleor github` command

```
saleor github [command]

Commands:
  saleor github login  Add integration for Saleor CLI

Options:
      --json     Output the data as JSON                               [boolean]
  -V, --version  Show version number                                   [boolean]
  -h, --help     Show help                                             [boolean]
```

### saleor github login

```
saleor github login

Add integration for Saleor CLI

Options:
```
